# Python-expense-automation

勤務表（Excel）から交通費明細書を自動生成するツールです。

手作業で行っていた入力業務を自動化し、
作業時間の削減と入力ミスの防止を目的としています。

## 背景

勤務表を見ながら交通費明細書を手入力しており、
時間がかかる・入力ミスが発生するという課題がありました。

## 解決方法

- 勤務表から出社日を抽出
- 日付から曜日を生成
- 交通費明細書へ自動入力

## 成果

- 手作業：約15分 → 数秒に短縮
- 入力ミスの削減

- ## 工夫した点

- 勤務表には出社以外のデータもあるため、「出社」の日のみ取得するようにしました。

- 日付の形式がバラバラだったため、正しく処理できるようにしました。

## 使い方

1. Excelファイルを配置
2. Pythonを実行
3. 明細書が生成される
   
## 使用技術
Python  
openpyxl
